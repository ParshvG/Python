git init -> to initalize folder
git clone https://github.com/ParshvG/FaceRecognition.git -> to clone github reposatry
git remote add origin https://github.com/ParshvG/FaceRecognition.git -> to connect with github reposatry
git status ->to check file status whether it is added or not (red-not add, green-added)
git add . -> to add all files
git commit -m "Your Message"-> to save all files
git push -> to push into remote browser
git checkout "branchname" -> to view/use specific branch 
git pull -> to pull data of specific branch
git merge "branchname" -> to merge data of specific branch
git branch -d "branchname" -> to delete local branch
git push origin --delete "branchname" -> to delete remote branch
git branch <branch_name>-> Creates a new branch.
git remote -v -> Lists all configured remote repositories.
git stash -> Temporarily saves changes that are not ready to be committed.
git stash apply -> Applies the most recent stash.
git config --global user.name "Your Name" -> Sets your name for commits.
git config --global user.email "you@example.com"-> Sets your email for commits.
